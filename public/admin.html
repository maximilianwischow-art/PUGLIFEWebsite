Raid Helper ↔ Warcraft Logs

This is the primary on-disk character roster (rh-wcl-character-links.json) for this deployment — same data as GET /api/wcl/guild/:guildId/characters. Attendance uses character names from logs; Events shows Raid Helper signups. For each person, enter their signup name and every log name they use (main + alts), comma-separated. If the log name matches signup, you can leave the second column empty.

Heuristic matcher (not an LLM): compares signup names from the latest Raid Helper posted events with characters seen across recent logs — exact match, prefix/alts, fuzzy spellings. By default signups are read from the last 6 events and log characters from the last 6 tracked raid reports (override with RH_WCL_LINK_RAID_HELPER_EVENTS / RH_WCL_LINK_WCL_REPORT_DETAILS). Review chips in Match source before relying on guesses.
