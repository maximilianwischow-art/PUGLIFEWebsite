1. Data We Process

We process Discord account data required for login (Discord user ID, username/global name, avatar hash), event participation information from Raid Helper, and MVP voting records on this website.

We use this data to provide member login, event signup functionality, and raid MVP voting features for the guild platform.

We set one necessary session cookie (`plb_session`) for authenticated login. This cookie is required for secure sign-in and protected actions such as event signup and voting.

Session and voting data are stored for operational purposes. Data may be removed on request where legally applicable.
